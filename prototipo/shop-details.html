Descrição Do Produto

Para gamers, streamers e multi-tarefas que precisam ter mais controle na ponta dos dedos, o HyperX Alloy Elite? 2 é o teclado certo para você. Com teclas de mídia exclusivas e um grande controle de volume, esse teclado gamer com todos os recursos está pronto para tudo desde edição de vídeo até assistir filmes. É construído com teclas mecânicas ultraconfiáveis HyperX ajustadas para velocidade e agilidade, para que você possa confiar no seu toque nas teclas. Os LEDs expostos das teclas combinam com as Capas de Tecla HyperX Pudding para dar à sua iluminação RGB um brilho extra comparado com as capas de tecla de cores sólidas. Personalize a barra iluminada exclusiva e crie perfis de iluminação com o software intuitivo HyperX NGENUITY para uma exibição brilhante de cores. Use o software para personalizar ainda mais o seu sistema com o Modo Jogo e macros personalizados. Uma entrada USB 2.0 fornece uma porta USB extra para suas necessidades em multitarefas. A estrutura em aço resistente do Alloy Elite 2 torna o teclado durável o bastante para trabalhar e jogar noite e dia.
